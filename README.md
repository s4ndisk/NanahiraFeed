# nanafeed.py
Python web-scraper for grabbing all posts/uploads from ななひら (Nanahira) across Spotify, Youtube and X (Formally Twitter)

### Content Types

#### Spotify
- Albums
- Singles
- Albums appeared on

#### Youtube
- Videos
- Streams

#### X (Formally Twitter)
- Tweets
- Retweets
- Replies (Soon, bugged atm)
